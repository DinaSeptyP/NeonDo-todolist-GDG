# **App Name**: NeonDo

## Core Features:

- Task Creation: Allow users to add new tasks with descriptions and due dates.
- Task Completion: Enable users to mark tasks as complete with a simple toggle.
- Priority Setting: Allow users to set priority levels for tasks (High, Medium, Low).
- Task Filtering: Implement task filtering by status (Complete, Incomplete, All).
- Thematic Organization: AI Tool that assists with the discovery of new tasks to enter, based on the user's personal schedule and historical patterns.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) to reflect the cyberpunk and futuristic aesthetic.
- Background color: Dark grayish-blue (#222F3A) to set a contrasting stage for the bright accents, and enhance legibility in a dark UI.
- Accent color: Neon Pink (#FF69B4) to add a jolt of contrasting energy for interactive elements.
- Body and headline font: 'Space Grotesk' for a futuristic, computerized feel.
- Use neon, glowing icons to represent task status and priority levels.
- Implement a clean, modular layout with clear sections for task lists and filtering options.
- Subtle glow effects and smooth transitions for a modern feel.
